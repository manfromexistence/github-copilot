API Documentation

Translate Text

POST /translate

Translates a given text to a target language.

Request Body (JSON)

{
  "text": "string (The text to translate)",
  "target_language": "string (e.g., 'es', 'fr', 'de')"
}

Response Body (JSON)

{
  "translated_text": "string (The translated text)"
}

Example Usage (curl)

curl -X POST "https://manfromexistence-api.vercel.app/translate" \
-H "Content-Type: application/json" \
-d '{
  "text": "Hello, world!",
  "target_language": "es"
}'

Text to Speech (TTS)

POST /tts

Converts a given text to speech and returns an MP3 audio file.

Request Body (JSON)

{
  "text": "string (The text to convert to speech)"
}

Response

An MP3 audio file (audio/mpeg).

Example Usage (curl)

curl -X POST "https://manfromexistence-api.vercel.app/tts" \
-H "Content-Type: application/json" \
-d '{
  "text": "Hello, this is a test."
}' -o output.mp3
